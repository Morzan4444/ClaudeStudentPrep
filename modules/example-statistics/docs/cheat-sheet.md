| Concept | Quick Reference |
|---|---|
| **Mean** | x̄ = Σxᵢ / n — sum all values, divide by count |
| **Variance (population)** | σ² = Σ(xᵢ − μ)² / N |
| **Variance (sample)** | s² = Σ(xᵢ − x̄)² / (n−1) — use n−1 for unbiased estimate |
| **Standard Deviation** | σ = √variance — same units as data |
| **Normal Distribution** | Symmetric bell curve; defined by μ and σ |
| **Empirical Rule** | 68% within 1σ, 95% within 2σ, 99.7% within 3σ |
| **Central Limit Theorem** | Sample means → Normal as n → ∞; works for n ≥ 30 |
| **p-value decision** | p < α → reject H₀; p ≥ α → fail to reject H₀ |
| **95% Confidence Interval** | x̄ ± 1.96 × (σ/√n) for large samples |
| **Correlation (r)** | −1 ≤ r ≤ 1; r > 0 positive, r < 0 negative, r = 0 no linear relationship |
| **Linear Regression** | ŷ = β₀ + β₁x; β₁ = slope, β₀ = y-intercept |
| **Standard Error** | SE = s / √n — standard deviation of the sampling distribution of the mean |
| **Z-score** | z = (x − μ) / σ — how many standard deviations from the mean |
| **Type I Error (α)** | Rejecting H₀ when it is true (false positive) |
| **Type II Error (β)** | Failing to reject H₀ when it is false (false negative) |
