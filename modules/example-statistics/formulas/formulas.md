**Arithmetic Mean**
```
x̄ = (x₁ + x₂ + ... + xₙ) / n = Σxᵢ / n
```
Explanation: Sum all observations and divide by the count to find the average value.

---

**Population Variance**
```
σ² = Σ(xᵢ − μ)² / N
```
Explanation: Average squared distance of each data point from the population mean.

---

**Sample Variance**
```
s² = Σ(xᵢ − x̄)² / (n − 1)
```
Explanation: Uses n−1 (Bessel's correction) to produce an unbiased estimate of population variance from a sample.

---

**Standard Deviation**
```
σ = √σ²    (population)
s = √s²    (sample)
```
Explanation: Square root of the variance; expressed in the same units as the original data.

---

**Z-Score (Standardisation)**
```
z = (x − μ) / σ
```
Explanation: Measures how many standard deviations a value x is above or below the mean.

---

**Standard Error of the Mean**
```
SE = s / √n
```
Explanation: Estimates the variability of sample means around the true population mean; decreases as sample size grows.

---

**95% Confidence Interval (large sample)**
```
CI = x̄ ± 1.96 × (s / √n)
```
Explanation: Range within which we are 95% confident the true population mean lies, based on a sample mean and standard error.

---

**Pearson Correlation Coefficient**
```
r = Σ[(xᵢ − x̄)(yᵢ − ȳ)] / √[Σ(xᵢ − x̄)² × Σ(yᵢ − ȳ)²]
```
Explanation: Measures the strength and direction of the linear relationship between two continuous variables; ranges from −1 to +1.

---

**Simple Linear Regression**
```
ŷ = β₀ + β₁x

β₁ = Σ[(xᵢ − x̄)(yᵢ − ȳ)] / Σ(xᵢ − x̄)²
β₀ = ȳ − β₁x̄
```
Explanation: Fits a straight line through data to predict y from x; β₁ is the slope (change in y per unit x) and β₀ is the y-intercept.
